# SEO & GEO Optimization Guide

Detailed optimization reference for both traditional search (SEO) and AI-powered answer engines (GEO).

## SEO Fundamentals

### Structure & Technical

- **Clear heading hierarchy** — H1 for title, H2 for main sections, H3 for subsections
- **Meta description** — 150-160 characters, include primary keyword, compelling hook
- **Internal linking** — Connect to relevant pillar content and related posts using descriptive anchor text (not "click here")
- **URL structure** — Short, descriptive slugs with primary keyword
- **Mobile optimization** — Fast load times, responsive design
- **Schema markup** — FAQPage, HowTo, Article schemas help both search and AI

### E-E-A-T Signals

- **Experience** — Share first-hand knowledge, real examples from your work
- **Expertise** — Demonstrate deep understanding, not surface-level summaries
- **Authoritativeness** — Build topic authority through consistent, comprehensive coverage
- **Trustworthiness** — Cite sources, maintain accuracy, transparent author attribution

### Keyword Integration

- Primary keyword in title, first 100 words, and 1-2 H2s
- Semantic variations throughout (AI understands synonyms)
- Question-based keywords match conversational search
- Write for humans first — avoid keyword density thinking

## GEO: Generative Engine Optimization

GEO optimizes content to be cited by AI systems (ChatGPT, Perplexity, Claude, Google AI Overviews). The goal: become the source AI trusts enough to quote.

### Platform-Specific Nuances

| Platform | Preferences |
|----------|-------------|
| **ChatGPT** | Encyclopedic content. Loves Wikipedia-style comprehensiveness and "definitive" guides. Authority matters heavily. |
| **Perplexity** | Rewards recency and community examples. Values "fresh" data over general principles. |
| **Google AI Overviews** | Prioritizes existing top-ranking content. Strong SEO = better AI Overview presence. |
| **Claude** | Values nuanced, well-reasoned content with clear structure. Less likely to hallucinate if logic is sound. |

**Universal truth:** Well-structured, authoritative, current content performs across all platforms.

### Core GEO Principles

**1. Answer-First Structure**

Put the direct answer in the first 40-60 words:

```markdown
Bad:  "Many people wonder about the best way to handle AI in schools.
       There are several perspectives to consider..."

Good: "Schools implementing assessment redesign see 40% fewer AI-related
       integrity issues. The shift from catching cheaters to designing
       better assignments is working."
```

**2. Fact Density**

Include specific statistics every 150-200 words. AI loves citing numbers.

- Bad: "Many teachers lack AI training"
- Good: "68% of urban teachers report receiving zero AI training since joining their schools"

**3. Quotable Statements**

Write clear, declarative sentences AI can lift directly. Front-load the insight:

- Bad: "It's worth considering that detection might not be effective..."
- Good: "Detection tools are unreliable — 94% of AI-generated work goes undetected."

**4. Structured for Extraction**

Pages with clear H2/H3/bullet structures are more likely to be cited. AI parses structure to find relevant chunks:

- Use descriptive headings (question-based work well)
- Break walls of text into scannable sections
- Use tables for comparisons
- Use numbered lists for processes
- Keep paragraphs focused on one idea each

**5. Source Transparency**

Cite explicitly. AI rewards transparent attribution:

```markdown
Bad:  "Research shows that..."
Good: "According to RAND's 2025 study, 54% of students now use AI for school."
```

### The FAQ Pattern (Highest GEO Performance)

Add FAQ section to pillar content. Format:

```markdown
## Frequently Asked Questions

### [Exact question someone would ask AI]

[Direct 2-4 sentence answer with specific data if available]

### [Another natural question]

[Direct answer, no hedging]
```

Implement FAQPage schema for these sections — it signals to both search and AI that this content directly answers questions.

## Combined SEO + GEO Checklist

### SEO
- [ ] Title includes primary keyword
- [ ] Meta description: compelling + keyword, 150-160 chars
- [ ] H1/H2/H3 hierarchy is clean and logical
- [ ] Internal links to 2-3 related posts (descriptive anchors)
- [ ] URL slug is short and descriptive
- [ ] Author byline and credentials visible

### GEO
- [ ] Core answer appears in first 50 words
- [ ] At least 3 statistics with named sources
- [ ] 2+ quotable thesis statements (clear, declarative)
- [ ] Question-based headings that match AI prompts
- [ ] Comparison table included (if relevant)
- [ ] FAQ section added (for pillar content)
- [ ] Updated/publish date visible

### Both
- [ ] Clean heading hierarchy
- [ ] Author byline + credentials
- [ ] FAQ section with schema
- [ ] Question-based headings
