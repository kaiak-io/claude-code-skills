# LinkedIn Platform Specs & Algorithm (2026)

Detailed platform specifications for LinkedIn content creation.

## Character Limits

| Content Type | Limit |
|-------------|-------|
| Text post | 3,000 characters |
| "See More" cutoff | ~210 characters |
| Article | ~125,000 characters |
| Poll option | 140 characters each |
| Video caption | 500-800 characters optimal |

## Optimal Post Length

- **Sweet spot: 1,300-1,400 characters** (highest engagement)
- **Minimum for algorithm favor: 500 characters** (under = interpreted as low-effort)
- **Over 2,000 characters: ~35% engagement drop**
- **Dwell time sweet spot: 30-45 seconds** reading time (~1,300-1,600 characters)
- B2B/technical audience: 1,800-2,000 characters is acceptable
- Creative professionals: 800-1,000 characters

## Algorithm Signals

### Dwell Time (Primary Signal)
- Posts with 61+ seconds dwell time: **15.6% engagement rate**
- Posts with <3 seconds: **1.2% engagement rate**
- 15+ seconds triggers reach multiplier
- <2 seconds signals low value regardless of other factors

### Engagement Hierarchy (Highest to Lowest Weight)
1. Share with commentary (highest endorsement signal)
2. Comment (5+ words) — worth 2x a like
3. Comments from industry experts: 5-7x more algorithmic weight
4. Multi-participant discussion threads: 5.2x more amplification
5. Saves/bookmarks (lasting value signal)
6. Reactions (helpful but least weighted)

### The Golden Hour
- Algorithm shows post to 2-5% of your network in first 60 minutes
- Engagement quality in this window determines platform-wide amplification
- Author responses within first 30 minutes: 64% more total comments, 2.3x more views
- Weak early engagement kills the post immediately

### What the Algorithm Penalizes
- Engagement bait ("Comment YES if...", "Tag 3 people")
- External links in post body (put in comments)
- Generic/low-effort posts under 500 characters
- Posting and not engaging with comments
- Identical content cross-posted from other platforms

## Posting Schedule

- **Best days:** Tuesday, Wednesday, Thursday
- **Best times:** 8-10 AM or 12-1 PM (audience's time zone)
- **Frequency:** 3-5 posts per week; quality over quantity
- 3-4 high-value posts/week beats 7 mediocre ones

## Content Format Performance

1. **Document/PDF Carousels:** 596% more engagement than text-only
2. **Multi-Image Posts:** 6.60% average engagement rate (highest single-metric)
3. **Long-Form Text (1,000-1,300 chars):** Most reliable format for consistent reach
4. **Native Video:** 20x more shares; 24% more likely to be boosted vs external links

## Image Specs

| Type | Dimensions |
|------|-----------|
| Landscape | 1200 x 628 px |
| Square | 1200 x 1200 px |
| Portrait | 1080 x 1350 px (4:5 ratio) |
| Carousel/Document | 1080 x 1080 px or 1920 x 1080 px |
| Format | JPG, PNG, GIF; max 5MB |

## Carousel/Document Specs

- Upload as PDF, PPTX, or DOCX (LinkedIn converts to swipeable slides)
- Optimal: 8-12 slides for educational content
- Max per slide: ~30 words
- First slide = hook/title, last slide = CTA
- Each swipe counts as an engagement signal
- 77% of technical audiences prefer carousels over other content types

## Video Specs

| Spec | Value |
|------|-------|
| Resolution | 1920x1080 ideal, min 640x360 |
| Ratio | 16:9 |
| Duration | Up to 10 minutes; sweet spot 30-90 seconds |
| File size | Max 5GB |
| Format | MP4, MOV, AVI |
| Captions | Essential — 85% watched without sound |

## Poll Specs

- Up to 5 answer options
- Each option: max 140 characters
- Duration: up to 2 weeks
- Non-anonymous: creators see who voted and their choices
- 6-12% engagement rates (highest-interaction format)
- Repurpose results into follow-up content
