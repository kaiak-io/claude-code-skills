# Visual Concepts by Pillar

Per-post visual concept designs for KAIAK blog featured images. Each concept describes the visual metaphor, color usage, and layout for one post.

## How to Use

When generating an image for a specific post, find its entry below and use the visual concept as the basis for the per-post request. Adapt as needed based on the actual article content.
